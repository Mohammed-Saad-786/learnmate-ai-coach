# 🎓 LearnMate AI Coach

LearnMate is an AI-powered career mentor that generates structured learning roadmaps for different tech domains.

## 🚀 Features

- AI career roadmap generation
- Domain-based learning paths
- Salary insights (India)
- Course recommendations
- PDF roadmap export
- Chat-style interface

## 🛠 Tech Stack

- Python
- Gradio
- Groq LLM
- FPDF
