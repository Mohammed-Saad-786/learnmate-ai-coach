It helps students and beginners identify the **right learning path, tools, and courses** based on their current skill level.

The system uses **LLM-powered guidance** to simulate a personalized **AI career mentor**.

# 🚩 Problem Statement

With the rapid growth of online learning platforms, thousands of courses are available across multiple domains such as **Web Development, AI, Cloud Computing, and Cybersecurity**.

However, many students face the following challenges:

- Difficulty identifying **where to start**
- Lack of **structured learning paths**
- Confusion between **multiple course options**
- No **personalized career guidance**

As a result, learners often feel **overwhelmed and lose direction** in their learning journey.

---

# 💡 Proposed Solution

**LearnMate AI Coach** solves this problem by acting as an **AI-powered career mentor** that:

- Generates **step-by-step learning roadmaps**
- Recommends **relevant courses and resources**
- Provides **role suggestions and salary insights**
- Adapts guidance based on **user skill level**
- Allows users to **download a PDF roadmap**

This creates a **guided learning experience** similar to having a real mentor.

# ⚙️ System Workflow

1️⃣ User enters a **tech domain** (example: frontend, AI, cloud)  
2️⃣ User selects their **skill level**  
3️⃣ AI generates a **personalized learning roadmap**  
4️⃣ System suggests **courses, roles, and tools**  
5️⃣ User can **download the roadmap as a PDF**
